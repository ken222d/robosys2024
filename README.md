# robosys2024
ロボットシステム学の練習リポジトリ

# plusコマンド
![test](https://github.com/ken222d/robosys2024/actions/workflows/test.yml/badge.svg)
![](https://img.shields.io/github/license/ken222d/robosys2024)
標準入力から読み込んだ数字を足す。

## 使い方


```bash
$ git clone https://github.com/ken222d/robosys2024
$ cd robosys2024
$ ./plus 1 2 3 4　　　　　　　　　　　　　　　　　　　#実行
10                                                    #結果
```


## 必要なソフトウェア
- Python
  - テスト済みバージョン: 3.7〜3.10

## テスト環境
- Ubuntu 22.04.5 LTS
